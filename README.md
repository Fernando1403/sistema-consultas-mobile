# Sistema de Consultas Mobile

## 👨‍💻 Autor
Fernando Gonzales  
RM: 555045  

## 📱 Sobre o Projeto
Este projeto consiste em um aplicativo mobile desenvolvido com React Native. O objetivo é criar um sistema de consultas que permita aos usuários realizar consultas de forma prática e rápida através do celular.

## 🚀 Status do Projeto
O projeto ainda está em desenvolvimento e atualmente possui a estrutura inicial, incluindo organização de arquivos, configuração do ambiente e primeiros componentes.

## 🛠️ Tecnologias Utilizadas
- React Native  
- TypeScript  

## 📂 Estrutura do Projeto
- `src/` → Código-fonte principal  
- `assets/` → Arquivos estáticos  
- `App.tsx` → Componente principal da aplicação  

## 📌 Objetivo
Facilitar o acesso a consultas por meio de um aplicativo simples, intuitivo e eficiente.
